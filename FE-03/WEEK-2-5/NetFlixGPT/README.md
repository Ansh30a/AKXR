# NetflixGPT
- Bun + Vite Setup \
```bun create vite@latest app-name```
- Configure tailwindcss \
```bun add -D tailwindcss @tailwindcss/vite```

## Features
- Login/Sign Up \
    - Sign in/ Sign up form
    - redirect to Browse 

    
- Browse \ (Authenticated User only)
    - Header
    - Main Movie (Hero)
        - Trailer
        - Title & Description
        - Play button && Info Button
    - Suggestions
    - Endless horizontally and vertically in different categories


- NetflixGPT \ (Authenticated User only)
    - Search Bar
    - Suggestions
